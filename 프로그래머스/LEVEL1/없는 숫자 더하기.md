## 없는 숫자 더하기

문제 : https://programmers.co.kr/learn/courses/30/lessons/86051

풀이

```java
import java.util.*;

class Solution {
    public int solution(int[] numbers) {
        List<Integer> nums = new ArrayList<>();
		for (int x : numbers) {
			nums.add(x);
		}
		
		int answer = 0;
		for (int i = 1; i <= 9; i++) {
			if (!nums.contains(i)) {
				answer += i;
			}
		}
        return answer;
    }
}
```
